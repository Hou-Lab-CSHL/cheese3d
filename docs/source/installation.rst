Installation
============

Requirements
------------

Cheese3D is a Python package with a few external dependencies. Our environment is managed by `Pixi <https://pixi.sh/latest/>`__. Install Pixi using (on macOS or Windows):

.. code-block:: bash

    curl -fsSL https://pixi.sh/install.sh | sh
